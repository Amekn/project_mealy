## What changed

- Linux is now Mealy's sole production operating-system target. The former
  macOS conversation preview is archived and is not shipped by this release.
- Ubuntu 24.04 and 26.04 LTS, Debian 13, and Fedora 44 are qualified on x86-64
  and ARM64. Arch Linux is qualified on x86-64.
- The same attested release payload is available as a rootless archive,
  Debian package, RPM, and, on x86-64, an Arch package.
- Clean distribution jobs build and exercise packages on every qualified
  Linux family, then the tag workflow downloads the public artifacts and
  repeats installation, service, health, backup, replay, and removal checks.
- Compatible derivatives are expected to work only when they preserve the
  documented glibc, filesystem, Bubblewrap, user-namespace, and systemd-user
  contract.
- The tokenless verified installer and provenance checks use the canonical
  `Amekn/mealy` repository identity after the repository rename.
