# Archived macOS preview

Mealy's macOS conversation-only preview ended with `v0.1.0`. It never provided the Linux worker
sandbox, governed tool execution, or a production support commitment. Starting with the next
release, active source qualification, packages, documentation, and publication target GNU/Linux
only.

This directory preserves the preview packager and native smoke procedures for historical review.
They are read-only evidence, are excluded from active CI and release assets, and are not maintained
or supported. Their original repository layout, exact source, native workflow, attestations, and
published ARM64/Intel preview archives remain permanently available from:

- [`v0.1.0` source](https://github.com/Amekn/project_mealy/tree/v0.1.0)
- [`v0.1.0` release](https://github.com/Amekn/project_mealy/releases/tag/v0.1.0)
- [`v0.1.0` release workflow](https://github.com/Amekn/project_mealy/actions/runs/29976463202)

The archived scripts intentionally retain their original paths and assumptions internally. Do not
run them from this relocated directory or interpret their presence as current platform support.
